# KBC Quiz Game 🎮

A console-based Kaun Banega Crorepati (KBC) quiz game built with Python.

## Features
- Multiple Choice Questions
- Prize Money System 💰
- Game Over on Wrong Answer ❌
- Winner Message 🏆

## Technologies Used
- Python

## How to Run

1. Download the project
2. Run `main.py`
3. Answer the questions and win prizes
